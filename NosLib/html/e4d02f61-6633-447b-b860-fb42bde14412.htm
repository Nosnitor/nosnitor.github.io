Welcome to the Nosnitor.Common project

This project provides functionality common to Nosnitor applications. Using these common methods and procedures provides consistency across all Nosnitor projects.

Getting Started

The following items are included in this project:

  • Nosnitor.Common.dll - The Nosnitor Common Library contains commonly used funcationality that is used across Nosnitor applications.

This project is included in the following packages:

  • Nosnitor.Common NuGet Package - The NuGet package is the easiest way to integrate NosLib into your project.
